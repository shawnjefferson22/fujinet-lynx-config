
Fujinet Lynx Config
===================

Press any button to bypass the splash screen quickly, or wait five seconds. If the Fujinet is already connected to Wifi, you'll go straight into the Hosts selection screen, otherwise the Wifi Scan and Select will start.

Common Input
------------
From any screen, these buttons will always work.
PAUSE: show Fujinet adapter config information, and current Fujinet data and time.
OPT2: start the wifi SSID scan and selection.

Common Text Entry:
-----------
Some functions require text input.  The field will turn grey, and you will be able to enter and edit text with the joypad, buttons and keys.  In text entry mode the following keys perform the functions below.

Joypad up/down: scrolls through the letters, numbers and characters.
Joypad left/right: moves the cursor appropriately. The cursor will not move past the leftmost or rightmost places.
A button: accepts the text as-is.
B button: cancels without accepting the text.
OPT1: insert a character
OPT2: delete a character
OPT1+OPT2: clear the text entry

Wifi Scan and Selection Screen:
-------------------------------
Pressing OPT2 will start a scan for wifi networks, which can take 5-10 seconds.  When complete, the top nine of the detected Wifi network is displayed, with highest strength networks topmost.

Joypad moves up and down the list.
A button: selects this wifi network (you'll be prompted to enter a password if required).
B button: rescans the wifi networks.
OPT2: exits the wifi selection screeen without changing networks.

Hosts Screen
------------
From this screen you can select and edit your list of hosts.

Joypad moves up and down the list.
A button: mounts a host and attempts to display the host's files.
B button: will allow you to edit that host entry.

File/Directory Selection Screen
-------------------------------
You can browse a hosts directories and files by using the joypad and buttons. If you are using a build for the Lynx GameDrive flash cart, you can also download and launch a Lynx ROM.  The following keys perform the functions below, in addition to the common functions:

Joypad up/down: move up and down in the list. The list will scroll to the next or previous page if at the top or bottom.
Joypad left: move to the previous page, or the next page if available.
A button: select directory or file.
B button: exit back to Hosts Screen.
OPT1: display information on directory or file.
